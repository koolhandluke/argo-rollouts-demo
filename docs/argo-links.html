Core UIs

Argo CD GitOps UI
https://localhost:8080
Username admin
Password run command below ↓
Port-forward
kubectl port-forward svc/argocd-server -n argocd 8080:443
Get password
kubectl get secret argocd-initial-admin-secret -n argocd -o jsonpath='{.data.password}' | base64 -d
Rollouts Rollouts Dashboard
http://localhost:3100/rollouts

No login required

Port-forward
kubectl port-forward svc/argo-rollouts-dashboard -n argo-rollouts 3100:3100

Or use: kubectl argo rollouts dashboard

Prometheus Metrics (canary analysis)
http://localhost:9090

No login required

Port-forward
kubectl port-forward svc/kube-prometheus-stack-prometheus -n monitoring 9090:9090

Canary analysis query: sum(rate(http_requests_total{status="2xx"}[2m])) / sum(rate(http_requests_total{}[2m]))

Namespaces at a Glance

All rollout statuses
kubectl get rollouts -A
All Argo CD apps (this demo)
kubectl get applications -n argocd | grep rollouts
All pods (non-running)
kubectl get pods -A | grep -v Running | grep -v Completed
rollouts-prod — watch canary
kubectl argo rollouts get rollout demo-app-demo-app -n rollouts-prod --watch
rollouts-prod — watch analysis
kubectl get analysisruns -n rollouts-prod -w
rollouts-staging — watch bluegreen
kubectl argo rollouts get rollout demo-app-demo-app -n rollouts-staging --watch
Image running in each env
for ns in rollouts-dev rollouts-staging rollouts-prod; do echo "$ns: $(kubectl get pods -n $ns -o jsonpath='{.items[0].spec.containers[0].image}' 2>/dev/null)"; done
